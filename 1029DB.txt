DB 설정

데이터 : 정보의 단편적인 조각
 
정보 : 데이터를 수집하고 가공해 의미를 부여한 결과물.

데이터 베이스 : 데이터를 효율적으로 저장하고 활용할 수 있도록
	       체계적으로 조직화한 데이터들의 집합체.

DBMS : (DataBase Management System)
-데이터 베이스 관리 시스템
-데이터 베이스를 효율적으로 관리하고 다루기 위한 소프트웨어
-DBMS 를 사용하여 데이터베이스의 생성 , 삭제 ,수정 등의 관리작업을 할 수 있다.
-Oracle, mySQL, mariaDB 등 DBMS 가 존재한다.

SQL ( Structured Query Language)
- DBMS 내부에서 데이터를 관리하기 위해 사용되는 표준화되는 언어.
- 대부분의 DBMS에서 일관되게 사용할 수 있다.

*데이터베이스의 구성 요소
1)데이터 베이스
-데이터들을 체계적으로 저장하고 관리하기 위해 사용하는 시스템

2)테이블
- 실제 데이터들이 저장되는 구성 요소
- 데이터들을 구조화해 저장해놓은 데이터들의 실제 집합

3) 레코드, 로우(Record,Row)
-테이블의 행

4) 필드, 컬럼( Field, Colume)
-테이블의 열 

DBMS -> 데이터 베이스 -> 테이블 -> 데이터

* MariaDB 자료형
-테이블을 생성할 때 각 필드에 들어갈 데이터들의 자료형을 정해줘야한다.
-자료형이 지정되면 해당자료형에 해당하는 데이터만 입력이 가능하다.

int : 정수
float: 실수
char : 고정 길이 문자열
         어떤 데이터를 넣든 데이터의 길이가 고정

varchar : 가변 길이 문자열
	문자열의 길이에 맞춰 데이터의 길이가 변한다
