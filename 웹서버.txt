웹 서버 (Web Server)
- 클라이언트의 요청을 받아 웹 페이지나 콘텐츠를 제공해주는 서버.

1) 정적 웹 페이지 
-> 클라이언트의 요청에 따라 제공되는 미리 만들어져 있는 웹페이지.

2) 동적 웹 페이지
-> 클라이언트의 요청에 따라 서버에서 동적으로 콘텐츠를 생성하고 제공하는
     웹 페이지
